Angular-Firebase Chat test Application
Chat Test Application – example of Angular client Single Page Application (SPA) with Cloud Firestore NOSQL database.
The Application was developed using
  • Angular 9 platform and framework
  • Angular Material 9 - Material Design components for Angular
  • Angular CLI
  • Cloud Firestore NOSQL database
  • HTML 5 and CSS 3, Style preprocessors:  SCSS, SASS, LESS
  • TypeScript and ECMAScript 2015 (ES6 JavaScript)
  • RxJS 6 (Reactive Extensions Library for JavaScript)

The Firebase Authentication provides backend services, SDKs to authenticate Users to the Application.
It supports Anonymous Authentication without requiring users to sign in first by creating temporary anonymous accounts.

Used Angular buildings blocks:
  • NgModules, Components, Directives, Pipes
  • Services and Dependency Injection (DI)
  • Reactive Forms and Forms Validators
  • RxJS Observables and Subjects
  • Router and Guards
The functions of the Application are described in the Specification.
