Overview

This is a test app to prove technical skill in Web UI and Firebase/Firestore.

We want to build a simple one-to-many chat interface. In this case we have many end users and 1 support person who chats with them, a Therapist.

The users and chat data are all stored in Firestore. All chat data should be loaded from Firestore. All updates to users and chat text should be tracked with a document or collection snapshot listener (see https://firebase.google.com/docs/firestore/query-data/listen)
